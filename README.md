  ポイント

- 粘性底層 (y⁺ < 5): u⁺ ≈ y⁺ を直接解く
  - 4.11（理論4.47）と良好な一致
- バッファ層 (5 < y⁺ < 30):
  - CFDがスムーズに遷移（理論解は補間のため参考値のみ）
- 対数則領域 (y⁺ > 100): 理論との誤差 < 1%
- 外層での飽和: u⁺が対数則から外れてチャネル中心に向かって飽和するのは物理的に正しい挙動（ウェイク領域）
